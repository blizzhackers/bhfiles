Hi. I wrote something for you guys.
It is custom script for faster leveling.

Ty for all guys what have tested that before posting here.

http://pastebin.com/CCty0HXd

What you need ?
Minimum 4 chars. One for diablo prep, one for nitha prep, one baalruner and ofc char what you want level.
you can use 3 extra chars: diablo helper, baalhelper, and shrine finder.
everychar what is not in config but use FastExp script is idler.

How to use ?
Save file to bots folder,
edit charnames inside file.
add to charconfig
Code: Select all
Scripts.FastExp = true;


Requirements
It is tested with latest kolbot with 1.5 core. And it is high recomended to use that version.
Latest Kolbot - CORE15
