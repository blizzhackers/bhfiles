Client -> Server 0x67

Thought I would share this packet I found while trying to find the bnet game creation stuff (in order to expand it and add more creation options), since it isn't listed in the list.

This packet is sent when you request to create a game on Bnet (at least for open, didn't test on closed yet)
Here's what I could figure out so far:
Code: Select all
struct D2PacketClt67      //size of 0x2E
{
   BYTE nHeader;         //0x00
   char szGameName[16];   //0x01
   BYTE nGameType;         //0x11
   BYTE nClass;         //0x12
   BYTE __013;            //0x13
   BYTE nDifficulty;      //0x14
   char szCharName[16];   //0x15
   WORD __025;            //0x25
   DWORD dwArenaFlags;      //0x27
   DWORD __02B;         //0x2B
};


The size is fixed, always 0x2E
